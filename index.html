// FAQ DATABASE

Apakah desain sudah termasuk koding (Frontend)?

Paket standar kami berfokus pada UI/UX (Figma). Namun, untuk paket Enterprise, kami menyediakan layanan *Developer Handoff* lengkap atau bisa menyertakan implementasi Frontend (React/Vue) dengan biaya tambahan.

Berapa lama estimasi pengerjaan?

Untuk MVP (3-5 screen) biasanya memakan waktu 3-5 hari kerja. Untuk platform trading lengkap (20+ screen) bisa memakan waktu 2-4 minggu tergantung kompleksitas fitur.

Apakah data strategi trading saya aman?

Sangat aman. Kami bersedia menandatangani NDA (Non-Disclosure Agreement) sebelum proyek dimulai untuk menjamin kerahasiaan algoritma dan strategi bisnis Anda.
